本文说明

本文介绍了下什么是批处理，批处理能够写什么，并且介绍了常用的批处理命令，特此记录一下。

什么是批处理

批处理，也称为批处理命令。顾名思义，批处理就是对某种对象进行批量的处理，通常被认为是一种简化的脚本语言，它应用于DOS与Windows系统中。
批处理扩展名为bat。

常用的批处理命令

@命令

@命令：执行窗口中不显示它后面的这一行的命令本身。

ECHO命令

1.echo off：关闭回显，不显示执行命令
2.echo on：显示执行命令。
3.显示信息
4.编辑文本文件

::命令

::注释命令

pause命令

pause命令：让当前程序进程暂停一下，并显示一行信息：请按任意键继续…

:和go命令

1.这2个命令是分不开的。
2.提前结束程序。goto end :end

%命令

%命令：参数

if命令

if输入判断

存在判断

结果判断

call命令

call命令：从一个批处理脚本中调用另一个批处理脚本

find命令

搜索命令，用来在文件中搜索特定字符串，通常也作为条件判断的铺垫程序

type a.txt :列出a.txt的内容
find还有其他几个参数v n i等

｜命令

1.help命令：显示所有非隐含命令或者是命令的帮助信息
2.more命令：逐屏或逐行显示
3.|的命令作用：就是让前一命令的输出当作后一命令的输入

>,>>命令

输出重定向命令，>会清除掉原有文件中的内容后把新的内容写入原文件

只会另起一行追加新的内容到原文件中，而不会改动其中的原有内容

<,>&,<&命令

管道命令，但一般不常用
1.<,输入重定向命令，从文件中读入命令输入，而不是从键盘中读入
2.>&将一个句柄的输出写入到另一个句柄的输入中
3.<&,将一个句柄的输入并将其写入到另一个句柄的输出中
